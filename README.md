# imgtobase64
将本地或web图片压缩并转为base64链接

支持的图像格式:

.jpg;.jpeg;.png;.gif;.ico;.bmp;.webp;.tif;.tiff;.svg

使用工具库:
- google libwebp-1.0.3
- pngquant.exe 2.12.0
- nconvert v7.25 + XnConvert 1.80 lib + rsvg-convert-dll-2.40.16
